# Навигатор по Старому кладбищу Таганрога

Веб-приложение для навигации по историческому некрополю с 136 объектами культурного наследия.

## Функции
- Интерактивная карта с метками объектов
- Поиск и фильтрация по 9 категориям
- Построение маршрутов от местоположения и от входа
- Маршруты по целым категориям
- Подробная информация об объектах (описание, фото)

## Технологии
- HTML5, CSS3, JavaScript ES6+
- Яндекс Карты API
- Адаптивная вёрстка

## Установка
1. Клонировать репозиторий
2. Открыть index.html в браузере
